- **分层架构**：模块由入口脚本（`run_*.py`）、任务定义层（Prompt/Validator 构造）和底层执行器（`mu.eval` CLI / `pytest`）组成。
- **数据驱动**：采用时间戳目录结构（如 `runs/YYYYMMDD-HHMMSS/`）隔离每次运行，通过 `summary.json` 和 `latest-summary.md` 聚合结果。
- **安全机制**：内置敏感信息脱敏逻辑（`_redact`）和密钥扫描（`scan_for_secret_patterns`），确保 API Key 不泄露至落盘产物。
- **扩展性设计**：支持通过 `agent_cmd_builder` 注入自定义代理逻辑（如 `fake_agent.py`），实现离线环境下的 DGM 流程验证。
