- **测试架构**：采用 `pytest` 异步驱动 (`asyncio_mode=auto`)，通过 `conftest.py` 注入全局 `ToolRegistry` 夹具。
- **隔离策略**：广泛使用 `FakeModel` 和脚本化消息（`FakeMessage/ToolCall`）模拟 LLM 交互，实现无网络、无付费的确定性闭环测试。
- **模块覆盖**：按功能域划分为 Agent 循环逻辑 (`test_agent_loop`)、工具执行与权限 (`test_tools`, `test_permission`)、自延伸与元工具 (`test_extension`, `test_metatool`)、以及评估与 DGM 流程 (`test_eval`, `test_dgm`)。
- **环境模拟**：利用 `tmp_path` 构建临时工作区，结合 `Session` 持久化与分支机制，验证状态管理与历史回溯能力。
