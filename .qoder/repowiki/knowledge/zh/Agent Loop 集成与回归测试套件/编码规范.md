- 在多个测试文件（如 `test_agent_loop.py`, `test_codeact.py`）中定义局部 `FakeModel` 类以模拟模型响应，避免依赖真实 API。
- 普遍使用 `tmp_path` 夹具创建隔离的临时目录作为 Agent 的工作区或 Session 存储路径，防止测试污染。
- 异步测试函数统一采用 `async def test_...` 命名规范，并直接 `await` Agent 或工具的执行结果。
- 通过断言 `agent.messages` 中的 `role` 序列（如 `system, user, assistant, tool`）来验证 Agent 循环的状态流转是否符合预期。
