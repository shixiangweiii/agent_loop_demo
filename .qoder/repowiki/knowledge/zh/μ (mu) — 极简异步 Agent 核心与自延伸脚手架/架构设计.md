- **分层架构**：采用 `mu/` 单包结构，逻辑上分为前端层（Textual TUI/Headless）、Harness 层（Agent Loop/事件流/上下文管线）、能力层（4 工具/Native Code-action）与 Transport 层（OpenAI SDK 适配）。
- **核心控制流**：基于 `asyncio` 的朴素 `while` 循环，无硬编码步数限制，通过模型产出的 Tool Calls 驱动状态流转，支持细粒度的 `terminate` 终止信号。
- **数据持久化**：实现 Tree Session 机制（JSONL 存储），支持从任意节点分支（Branching）、回溯（Rewind）及侧分支摘要注入，确保上下文的可观测性与可恢复性。
- **扩展协议**：定义基于 JSONL stdin/stdout 的子进程通信协议，包含 Manifest 声明、Tool Schema 注册及 State 持久化，实现“代码即动作”的自延伸闭环。
