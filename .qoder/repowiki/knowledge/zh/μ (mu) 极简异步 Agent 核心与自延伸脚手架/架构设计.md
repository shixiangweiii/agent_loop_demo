- **统一事件流驱动**：所有子模块（核心循环、TUI、评测）均基于同一套异步事件流进行通信与状态同步，实现了解耦的观察者模式。
- **标准化会话契约**：采用 JSONL 格式的树状会话存储作为跨模块共享的状态边界，支持分支、续跑及侧分支摘要，确保了核心逻辑与评测框架的状态一致性。
- **子进程隔离扩展**：通过定义严格的 JSONL 协议，将自延伸工具的执行环境隔离在独立子进程中，保障了核心 Agent 的稳定性与安全边界。
- **模块化启动编排**：`pyproject.toml` 定义了统一的入口点 `mu.cli:main`，根据参数动态装配核心循环、TUI 界面或评测 Runner，实现了单一代码库下的多形态运行。
