- **分层结构**：模块按评测阶段划分为基础评测（`run_basic_eval.py`）、回归适配（`run_regression_eval.py`）和 M4 综合回归（`run_m4_regression_eval.py`）。
- **任务驱动设计**：采用“Prompt 注入 + Workspace 隔离 + Validator 校验”的闭环模式，每个任务在独立目录中运行并生成 `summary.json/md`。
- **产物归档机制**：通过 `dgm-smoke` 等子模块实现候选配置（Candidate）的隔离测试，并将结果归档至 `archive/` 目录，支持版本化对比。
- **安全与审计**：内置敏感信息扫描（`scan_for_secret_patterns`），确保 API Key 等机密数据不泄露至评测报告中。
